一个暗箱装有12个编号从1到12的乒乓球，甲、乙、丙三人轮流从暗箱中摸球，每人每次摸一个球且不放回。将所有球摸完后，三人所摸出的球上的编号之和相等，并且甲摸出了1号球和3号球，乙摸出了6号球和11号球。丙摸出的球编号最大为多少？
数列思维：球的编号总和78 平分26 其中：甲摸出了 1 号和 3 号，剩余两球编号之和=26−1−3=22，只能为 10 号和 12 号；乙摸出了 6 号和 11 号，剩余两球编号之和=26−6−11=9，故乙摸出剩余两球中的最大编号不能为 9

  
一个公比为2的等比数列，第n项与前n－1项和的差等于5，则此数列前4项之和为：
设首项为 ( a ) ，则有 ( $a \times 2^{n - 1} - \frac{a(1 - 2^{n - 1})}{1 - 2} = 5$ ) ，解得 ( a = 5 ) ；前四项分别为 5、10、20、40，和为 75。 故正确答案为 D。


某科研单位欲拿出一定的经费奖励获奖的科研人员，第一名可得到全部奖金的一半多1万元，第二名可得到剩余的一半多1万元，以此类推都得到剩余奖金的一半多1万元，若到第七名恰好将奖金分完，则该单位需要拿出奖金多少万元：
1.第七名拿之前还剩奖金为2万元，由此倒推第六名至第一名拿之前的奖金数分别为6、14、30、62、126、254万元。
2.设该单位拿出奖金为  x  万元，第一名得到 ( ==a== = $\frac{x}{2} + 1$ ) ，第二名得到 $(\frac{1}{2}(x - a)$ + 1 = $(\frac{x}{2} + 1) - \frac{a}{2} = \frac{1}{2}a )$ ，第三名得到 ( $\frac{1}{2}[x - (a + \frac{1}{2}a)] + 1 = (\frac{x}{2} + 1) - \frac{3}{4}a = \frac{1}{4}a ) ，(\cdots)$ ，由此可知，七名科研人员得到的奖金数是首项是 2，公比为 2，项数是 7 的等比数列。由第七名恰好分完，可知第七名得到奖金 2 万元。由等比数列求和公式可知，该单位拿出奖金为 ( $\frac{2 \times (1 - 2^7)}{1 - 2} = 254$ ) 万元。 故正确答案为 B。
